Fatal error: 'Table mysql.user doesn't exist'

Pour une raison qui m’est encore inconnue, la table mysql.user de mon serveur local a disparu. Après une bonne heure de recherche, je suis tombé sur ce post Stackoverflow détaillant le schéma de la table.

Pour résoudre le problème, allez dans le dossier « bin » de votre installation MySQL, ouvrez une invite de commande et exécutez :

mysqld --console --bootstrap --verbose

Dès que l’exécution bloque, collez le code ci-dessous et patientez. Si ça ne se termine pas, faites un gros CTRL+C pour tuer le processus.

C’est presque terminé ! En effet, il reste à recréer l’utilisateur par défaut MySQL.

Pour cela, créez un fichier « script.sql » (ou autre) avec ces instructions à l’intérieur :

CREATE USER 'root'@'%' IDENTIFIED BY ''; GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' WITH GRANT OPTION;

Puis exécutez :

mysqld --init-file=<CHEMIN VERS script.sql>

Coupez le processus (CTRL+C) et lancez Wamp ou tout autre environnement.

En espérant vous avoir fait gagner l’heure et demie que j’ai perdu 🙂
