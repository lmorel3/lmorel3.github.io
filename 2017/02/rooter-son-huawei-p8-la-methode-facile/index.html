Rooter son Huawei P8 : la méthode facile

February 4, 2017 3-minute read

Après quelques déboires lors de mes tentatives de root de mon Huawei P8, tout fonctionne. C’est pourquoi j’ai décidé de partager la (seule) méthode qui a fonctionné. C’est parti !

Étape 1 : Le compte Huawei ID

  1. Ouvrir l’application Huawei ID puis dans dans « Settings », cliquez sur « Créer compte et se connecter »
  2. Valider le compte
  3. Rendez-vous sur https://emui.huawei.com/en/plugin/unlock/index
  4. Pas le choix, vous devez « Accepter les CGU »
  5. Remplissez le formulaire. Les informations sont trouvables dans « Paramètres > Informations sur le téléphone »
Cependant, pour avoir votre « Serial number », vous devez composer le  `*#*#1357946#*#*`

Vous obtiendrez un message du type : `Unlocking : 6-------2`
  1. Gardez dans un coin de votre ordinateur le code permettant de débloquer le téléphone

Étape 2 : ADB, le débogage, toussa

Si ADB est déjà présent sur votre ordinateur, passez à l’étape 3.

  1. Téléchargez ADB Installer sur XDA-Developers (lien direct pour la version 1.4.2)
  2. Lancez l’exécutable en mode Administrateur
    1. « Install ADB and fastboot » -> Y
    2. « Install ADB System Wide » -> n
    3. « Install drivers » -> Y
  3. Activez maintenant le débogage USB (source: android-recovery.fr)

Etape 1. Débloquer le mode développeur Trouvez les « Réglages » et tapez-le. Cliquez sur « A propos de Téléphone», tapez « Numéro de version » sept fois jusqu’à obtenir une note « Vous êtes sous le mode développeur ».

Etape 2. Activer le débogage USB Android On retourne aux « Réglages » pour entrer les « Options pour les développeurs », puis on peut cochez « Débogage USB ».

Étape 3 : Déblocage

  1. Démarrez le téléphone en mode fastboot (maintenir « vol down » et « power  » pendant 10s)
  2. Lancer l’invite de commandes et exécutez fastboot devices
Vous verrez apparaître votre téléphone : `S\***\***\***\***\****        fastboot`
  1. Exécutez « fastboot oem unlock CODE_DE_L_ETAPE_1 »
```
...
OKAY [  1.554s]

finished. total time: 1.559s
```

Sur le téléphone, on voit apparaître « Phone unlocked » : le bootloader est débloqué !
  1. Tapez fastboot reboot pour redémarrer le téléphone

 

Étape 4 : Flash du recovery

  1. Télécharger TWRP pour P8 (lien pour la version 3.0.2.0) et copiez-le dans un répertoire quelconque
  2. Démarrer à nouveau le téléphone en mode fastboot, et lancer l’invite de commandes
  3. Déplacez-vous dans le dossier dans lequel vous avez placez le fichier contenant le recovery
  4. Exécutez « fastboot flash recovery .\KangV-HRT-TWRP-3.0.2.0.img »
```
target reported max download size of 471859200 bytes

sending 'recovery' (26120 KB)...

OKAY [  0.559s]

writing 'recovery'...

OKAY [  1.519s]
```

finished. total time: 2.088s
  1. Redémarrez : fastboot reboot

 

Étape 5 : Installez le binaire SU

/!\ Si TWRP demande si on veut installer SuperSU : Refuser !!

  1. Téléchargez SuperSU (lien pour la version 2.67) et copiez-le dans votre téléphone
  2. Démarrez cette fois-ci en mode recovery (« adb reboot recovery » dans l’invite de commandes ou alors démarrez avec « vol up + power »)
  3. Allez dans le menu « Advanced » et cliquez sur « Terminal », puis sur « OK »
  4. Tapez la commande : echo SYSTEMLESS=true >> /data/.supersu
  5. Allez dans « Install » puis choisissez le fichier et validez (« Swipe »)
  6. Redémarrez

/!\ Si TWRP demande si on veut installer SuperSU : Refuser !!

 

Ça y est ! Si tout c’est bien passé, vous avez désormais les privilèges d’un super-utilisateur et vous pourrez profiter d’un accès total à votre téléphone.  Super, non ? 🙂
